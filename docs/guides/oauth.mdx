# OAuth

By default the Panel contains the following OAuth providers:

* Facebook
* [X (Twitter)](https://developer.twitter.com/en/apps)
* LinkedIn
* [Goggle](https://console.developers.google.com)
* [Github](https://github.com/settings/developers)
* Gitlab
* Bitbucket
* Slack
* Authentik
* [Discord](https://discord.com/developers/applications)
* [Steam](https://steamcommunity.com/dev/apikey)
* [WHMCS](https://docs.whmcs.com/OpenID_Connect)

Additional providers can be added with plugins. (once the plugin system has been implemented)
